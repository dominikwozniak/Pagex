# :gem: Pagex

![npm](https://img.shields.io/npm/v/npm)
![GitHub last commit](https://img.shields.io/github/last-commit/dominikwozniak/Pagex-nextjs)

Pagex is a complete web app for creating static landing pages  :cyclone: 

## Technologies

* `NextJS`
* `MaterialUI`
* `Hooks`
* `SCSS`
* `Axios`
* `Styled Components`
* `JWT authentication`
